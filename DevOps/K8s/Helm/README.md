# Helm
## Prerequisite
* docker
* kubectl
* (kubeadm + kubelet) or (minikube)
## Install Helm
```sh
$ wget https://github.com/helm/helm/archive/v3.0.0-alpha.1.tar.gz
$ tar -zxvf helm-v3.0.0-alpha-1-linux-amd64.tar.gz
$ cd linux-amd64/
$ cp helm /usr/local/bin/
```
